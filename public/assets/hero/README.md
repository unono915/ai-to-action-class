# 시작 화면 메인 이미지

현재 시작 화면은 다음 이미지를 사용한다.

```text
public/assets/hero/start-hero-v2.webp
```

- 크기: 1536×864
- 비율: 16:9
- 이미지에 글자·로고를 넣지 않는다.

발표자인 남성 교사가 동료 교사들과 이미지 입력 → AI 판단 → 수업 행동의 흐름을
설계하는 장면이다. 발표자 얼굴은 사용자가 제공한 증명사진의 특징을 참고했으며,
증명사진 원본은 웹 자산에 포함하지 않는다. 이미지 파일을 불러오지 못하면 깨진
이미지나 이모지 대신 같은 흐름을 텍스트 카드로 표시한다.
