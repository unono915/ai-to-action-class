# 사이트 문구 초안

Claude Code는 아래 문구를 기준으로 실제 화면을 구성한다.
표현을 짧게 다듬을 수 있으나 교육적 의미를 바꾸지 않는다.

## 1. 시작하기

### 제목

판단하는 AI, 움직이는 수업

### 행사 정보

2026 AI·디지털러닝 콘페스타

2026년 7월 24일 · 마곡 코엑스

경문고등학교 지윤호

### 한 줄 설명

Teachable Machine과 Gemini Gems로 AI의 판단을 수업 활동과 문제 해결로 연결해 봅니다.

### 오늘 가져갈 것

- 손 모양 이미지 분류 모델
- 판단 결과에 따라 텍스트와 이미지를 출력하는 행동 연결 경험
- 내 교과 AI 모델 활용 아이디어
- 학생의 개념 이해를 점검하는 나만의 Gem 초안

### 준비 확인

- Chrome 브라우저
- 웹캠
- Google 계정 로그인
- 학생들이 자주 어려워하는 교과 개념 하나

## 2. 왜 이런 수업인가

### 장면 1

학생들은 묻습니다.

“이걸 왜 배우나요?”
“배워서 어디에 쓰나요?”

### 장면 2

배운 지식은 이해에서 끝나는 것이 아니라,
학생의 삶과 문제 해결로 확장될 때 살아납니다.

### 장면 3

AI 시대에 필요한 것은 도구 사용법만이 아닙니다.

- AI를 사용하는 방법
- AI가 판단하는 원리
- 판단 결과가 활용되는 구조
- AI의 판단을 검토하는 태도

### 장면 4

분류 모델 수업이 여기서 끝난다면

입력 → 모델 판단 → 레이블

학생은 “AI가 이미지를 구분하는구나.”라고 이해하고 넘어갈 수 있습니다.

### 장면 5

판단을 행동으로 연결하면

입력 → 모델 판단 → 레이블 → 텍스트·이미지·소리·움직임

학생의 질문은 “이걸 어디에 쓸 수 있을까?”로 바뀝니다.

### 장면 6

모델을 훈련시키는 일과
판단 결과에 따라 행동을 설정하는 일은 별개의 단계입니다.

### 장면 7

AI가 인간의 판단 일부를 대신할 수 있다는 것과
모든 상황에서 인간보다 더 잘 판단한다는 것은 다른 문제입니다.

### 장면 8

판단하기 전에 묻습니다.

- 데이터가 충분한가?
- 레이블 기준이 명확한가?
- 실제 환경에서도 작동하는가?
- 틀렸을 때 위험하지 않은가?
- AI가 판단해도 되는 문제인가?

### 장면 9

이미지 분류 수업 아이디어를 검토하는 네 질문

1. 보이는가?
2. 나눌 수 있는가?
3. 모을 수 있는가?
4. 판단 뒤 행동할 수 있는가?

## 3. 분류 모델 만들기

### 도입

공통 실습으로 O, X, 배경을 구분하는 손 모양 이미지 분류 모델을 만듭니다.

### 데이터 수집 안내

- 손 위치를 조금씩 바꿔 주세요.
- 카메라와의 거리를 바꿔 주세요.
- 손의 각도를 바꿔 주세요.
- 각 레이블의 데이터 수를 비슷하게 맞춰 주세요.
- 배경 레이블에는 손동작이 없는 화면을 담아 주세요.

### 테스트 질문

모델은 정말 손 모양을 보고 있을까요?
혹시 배경이나 손의 위치를 더 많이 보고 있지는 않을까요?

### 파일 업로드 확장

웹캠으로 직접 촬영하는 것 외에 이미지 파일을 업로드하여 학습할 수도 있습니다.
암석, 식물, 미술 작품, 악기처럼 현장에서 바로 준비하기 어려운 대상을 수업에 활용할 수 있습니다.

## 4. 판단을 행동으로

### 핵심 문구

모델은 판단합니다.
이제 그 판단에 따라 무엇을 할지 정해 봅니다.

### 실습 설명

1. Teachable Machine에서 모델을 내보냅니다.
2. 생성된 모델 주소를 복사합니다.
3. 행동 연결 사이트를 엽니다.
4. 모델 주소를 입력합니다.
5. 레이블마다 출력할 텍스트와 이미지를 설정합니다.
6. 웹캠으로 판단 결과와 행동을 확인합니다.

### 확장 문구

오늘은 텍스트와 이미지를 출력했지만,
같은 판단 결과를 음성, 게임, LED, 모터, 로봇의 움직임과 연결할 수도 있습니다.

## 5. 내 교과로 확장하기

### 핵심 문구

모델을 만드는 것보다 중요한 것은
그 모델의 판단을 내 수업의 어떤 활동과 연결할 것인가입니다.

### 아이디어 공식

교과 개념
+ 관찰 가능한 대상
+ 명확한 레이블
+ 수집 가능한 데이터
+ 판단 뒤 행동

### Gem 안내

교과별 아이디어 코치 Gem과 대화하며
내 교과에서 이미지·자세·음성 모델을 활용할 수 있는 수업 아이디어를 구체화합니다.

## 6. 나만의 Gem 만들기

### 도입

모든 학생 옆에 한 명의 교사가 붙어
설명을 듣고 추가 질문을 던지기는 어렵습니다.

Gemini Gems를 잘 설계하면
학생이 자기 말로 개념을 설명하도록 돕는 개별 질문 파트너로 활용할 수 있습니다.

### 실습 목표

정답을 바로 알려주는 챗봇이 아니라,
학생의 설명을 듣고 빠진 부분을 질문하는 개념 체크 도우미를 만듭니다.

### 좋은 지침의 구조

- 역할
- 대화 목표
- 대화 순서
- 확인할 핵심 요소
- 하지 말아야 할 행동
- 힌트 제공 방식
- 마지막 산출물
- 첫 질문

## 7. 자료 기반 Gem

### 선택 심화

수업 자료나 Notebook 기반 자료를 Gem의 지식으로 연결하면
제공한 자료를 우선 근거로 답하도록 유도할 수 있습니다.

### 주의

자료를 연결하면 오류 가능성을 줄이는 데 도움이 되지만,
AI의 답변이 완전히 무오류가 되는 것은 아닙니다.

학교 계정과 관리자 설정에 따라 기능이 보이지 않을 수 있습니다.

## 8. 공유와 마무리

### 실천 계획

나는 __________ 수업에서
__________ 모델 또는 Gem을 활용하여
학생들이 __________하도록 해보겠습니다.

### 마무리 문구

오늘 중요한 것은 도구를 완벽히 익힌 것이 아닙니다.

학생이 배운 지식을 자기 삶과 문제 해결로 확장하도록
AI의 판단과 행동을 수업 안에서 어떻게 연결할지 생각해 보는 것입니다.

내 수업의 작은 개념 하나,
작은 질문 하나부터 시작할 수 있습니다.
