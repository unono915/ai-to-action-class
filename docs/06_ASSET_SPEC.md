# 최종 메인 이미지 및 화면 자산 명세

## 1. 메인 이미지

### 파일 경로

```text
public/assets/hero/start-hero-v2.webp
```

### 비율과 크기

- 16:9
- 현재 크기 1536×864
- WebP, 약 85KB

### 시각적 내용

- 왼쪽: 손동작 이미지 입력 카드
- 중앙: 교사와 추상적인 AI 판단 노드
- 오른쪽: 말풍선·이미지·아이디어·수업 행동 카드
- 주변: 노트북으로 함께 설계하는 성인 교사
- 밝고 현대적인 교육 기술 분위기
- 파란색·인디고·보라색 계열
- AI나 로봇보다 교사가 흐름을 설계하는 장면을 중심에 둠
- 이미지 안에는 한글·영문 문구와 로고를 넣지 않음

### 화면 적용

- 시작 화면 hero
- 1단계 발표 모드 표지
- `object-fit: cover`
- 중요한 피사체가 중앙 안전 영역에 위치
- 모바일 crop에서도 중앙 AI와 판단→행동 흐름이 남아야 함

## 3. 자산 저작권

- 사용자가 제공하거나 직접 제작한 자산만 최종 포함
- 외부 검색 이미지를 임의로 저장소에 포함하지 않음
- Google 서비스 로고와 화면은 교육 안내 범위에서 캡처를 사용하되 공식 서비스로 오인시키는 로고 재가공은 피함
